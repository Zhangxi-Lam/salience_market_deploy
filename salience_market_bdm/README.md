# otree_multiple_asset_market

This experiment is a multiple-asset CDA market implemented using [oTree Markets](https://github.com/Leeps-Lab/otree_markets). To install, follow the installation instructions for oTree Markets [here](https://github.com/Leeps-Lab/otree_markets/wiki/Installation). Then clone this repo into your oTree project folder and add the config dict to `SESSION_CONFIGS` in settings.py.
